# IT Support Training

This section contains my structured, hands-on training in IT support fundamentals. It focuses on building practical skills through labs, troubleshooting scenarios, and system configurations.

## 📚 Coverage
- Hardware and system fundamentals
- Windows and Linux operating systems
- Troubleshooting methodologies
- Networking basics (IP, DNS, DHCP)
- Security fundamentals
- Basic automation and scripting

## 🛠 Approach
Each module includes:
- Step-by-step lab activities
- Command-line practice
- Screenshots and outputs
- Issue resolution and fixes
- Key learnings and reflections

## 📂 Structure
- Organized by topic and progression
- Includes notes, lab exercises, and supporting files

## 🎯 Objective
To develop real-world IT support skills by simulating common technical issues and applying structured troubleshooting techniques.

## 📌 Note
All work is performed in a controlled lab environment using virtual machines and real system configurations.
